# MedRAG: AI-Powered Medical Chatbot

## Description
This project is a medical chatbot using Retrieval-Augmented Generation (RAG).

## Features
- Medical Q&A system
- FAISS-based search
- GPT-based response generation

## Tech Stack
- Transformers (GPT)
- FAISS
- Sentence Transformers
- Streamlit

## Structure
Organized into modular folders for scalability.

## Disclaimer
This chatbot provides general medical information only.
